Dicio

The Wireless Sensor Wizards (@kamladi, @dansantorojr, and @adamselevan) designed and implemented a power outlet wireless network called Dicio (latin for ‘to control’). Users are able to remotely access information about the current and past power consumption and environmental data in their home from anywhere there is an internet connection. Based on this information, users are able to make decisions about the current state of their home and control the power to individual outlets.

Features

Dicio consists of two types of nodes communicating on an ad-hoc distributed network of sensing and control modules. Each node uses the CMU developed FireFly platform for radio hardware and communication, and custom hardware peripheral hardware was developed to sense power consumption, sample environmental data, and actuate the user’s requests. The gateway node is the the method by which the network control modules receive commands and pass information back to the web server.

The user interfaces with this web server via a mobile application to view data and control individual outlets. The mobile applications gives the user the opportunity to view the current sensor readings and control the power to any individual outlet immediately or with scheduled events. This gives the user the ability to turn off a lamp that was left on after a busy morning or randomize the lighting pattern in their home during a week of vacation.

Tech Stack

  • Hardware: FireFly wireless sensor nodes, along with a custom sensor and actuation board
  • RTOS: Nano-RK
  • Server: Node.js, MongoDB
  • App: React Native

Use Cases

Simple Use

At the most simple level, users can use a Dicio outlet to extend their outlet existing outlets with the added feature of remote actuation. Usuers can use the app, view the current status of their outlets, and toggle their status from anywhere.

Monitoring Your Home

One of extra benefits of the Dicio smart outlets is the power and ambient monitoring of users homes. Dicio gives you insight into the power use, temperature, and light in wherever there is a Dicio outlet. From the app, users can view current and past data, allowing them to view trends and make intelligent decisions about their power use.

Creating Events

Dicio smart outlets allow users to combine the sensing and actuation abilities of their outlet by creating and scheduling events. Events can be triggered based calendar time or any of the sensors that Dicio supports. Users can turn on their lamp in the morning as an alarm or turn on a fan when a room gets warm.

Group Sensing / Actuation

An added feature of Dicio smart outlets allows users to create events for groups of outlets for actuation. That is, turn on all of the lights if any of the outlets detect dusk approaching.

Future Uses

In the future, Dicio will support vacation mode, where events are created randomly over the course of your leave from home. In this way, the lights in users’ homes can be randomly turned on and off, made to look like real activity. In this way, users’ homes can be more secure than ever.
